# TextureGenerator
An image generator, written in Python 3 with Pygame

Controls:

    Esc.....Exit
    
    1.......Generate Image
    2.......Save Image
    0.......Reset Selected Value
    
    G.......Toggle Color
    M.......Toggle Mode
    R.......Randomize Seed
    
    W.......Menu Selection
    S.......Menu Selection
    A.......Decrement Selection
    D.......Increment Selection
